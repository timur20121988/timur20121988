## ⚡ TSHK IDE

**TSHK IDE** — это моя собственная IDE для кодинга с искусственным интеллектом.  
В отличие от Cursor, Claude Code или Codex — **ты сам выбираешь какой AI использовать**.

### 🔥 Возможности TSHK IDE

| Функция | Описание |
|---------|----------|
| 🤖 **Любой AI провайдер** | Anthropic Claude, OpenAI GPT, DeepSeek, Ollama, свой сервер |
| 🔗 **Свой эндпоинт** | Вставь любой URL — свой API, локальный сервер, прокси |
| 🔑 **Свой API ключ** | Полный контроль над доступом |
| 🎯 **6 режимов** | Normal, Turbo, Deep, Creative, Full Bypass, Agent |
| 📁 **Автосканирование** | Сама находит и читает файлы с твоего компьютера |
| 🪟 **Настоящий установщик** | Inno Setup, Windows программа, uninstall |
| 💾 **Сохранение** | История чата и настройки не пропадают |

### 🎯 Режимы AI

```
⚡  Normal      — Стандартный сбалансированный режим
🔥  Turbo       — Быстрый, лёгкий, минимум объяснений
🧠  Deep        — Глубокий анализ кода, edge cases, security
🎨  Creative    — Креативные решения, несколько подходов
🌀  Full Bypass — Без ограничений — максимальная мощность
🤖  Agent       — Автономный агент: планирует, пишет, тестирует
```
